# Inventory App

Мобильное приложение для инвентаризации склада с использованием сканирования штрих-кодов.

## Функциональность

- 📷 Сканирование штрих-кодов с помощью ML Kit и CameraX
- 📦 Автоматическое определение товаров и создание новых
- 🗂️ Управление группами инвентаризации
- 📊 Динамический список с автоматической сортировкой
- 📤 Отправка отчетов в Telegram
- 💾 Локальное хранение данных с Room

## Технологии

- Kotlin
- Jetpack Compose
- MVVM архитектура
- Room Persistence Library
- CameraX
- ML Kit Barcode Scanning
- Dagger Hilt для DI
- Kotlin Coroutines с Flow

## Сборка

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Дождитесь завершения синхронизации Gradle
4. Запустите приложение на устройстве или эмуляторе

## Требования

- Android 7.0 (API 24) или выше
- Разрешение на использование камеры
